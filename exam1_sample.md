## Problems

### Problem 1: Sum of Numbers (20 points)

Write a program that asks the user to enter 5 numbers and displays their sum.

**Example:**

```text
Enter number 1: 10
Enter number 2: 20
Enter number 3: 15
Enter number 4: 5
Enter number 5: 30
Sum: 80
```

***

### Problem 2: Grade Checker (20 points)

Write a program that asks for a test score (0-100) and displays the letter grade.

* 90-100: A

* 80-89: B

* 70-79: C

* 60-69: D

* Below 60: F

**Example:**

```text
Enter score: 85
Grade: B
```

***

### Problem 3: List Operations (20 points)

Write a program that:

1. Creates a list of 5 numbers

2. Finds and displays the largest number

3. Finds and displays the smallest number

4. Displays the average

**Example:**

```text
Numbers: [10, 25, 15, 30, 20]
Largest: 30
Smallest: 10
Average: 20.0
```

***

### Problem 4: Dictionary Lookup (20 points)

Write a program that:

1. Creates a dictionary of 5 countries and their capitals

2. Asks the user to enter a country name

3. Displays the capital if found, or "Not found" if not

**Example:**

```text
Enter country: France
Capital: Paris
```

***

### Problem 5: String Reversal (20 points)

Write a program that asks the user to enter a word and displays it reversed.

**Example:**

```text
Enter a word: hello
Reversed: olleh
```
